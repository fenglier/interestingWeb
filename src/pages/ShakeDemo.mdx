# 抖动效果 Shake

抖动效果常**引起用户注意或提示错误**，是“负反馈”的常见表现

> 它是使用translate和transform实现的。

```scss
@keyframes shake {
  0% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-3px);
  }
  100% {
    transform: translateY(0);
  }
}
```
